# FaccioCtrl - Lead Capture

Landing page de captura de leads para controle de ordens de facção de costura.
Stack: React 19 + Vite 6 + TypeScript + Tailwind CSS v4 + Motion.

## Rodando localmente

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run preview
```

## Deploy na Vercel

1. Suba este repositório no GitHub (veja instruções no chat).
2. Em vercel.com, clique em "Add New Project" e importe o repositório.
3. A Vercel detecta automaticamente o framework "Vite". Não é necessário mudar nada.
4. Clique em "Deploy".
